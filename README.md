# Ayana App Front Page
***Project Info***

- Flutter v.3.10.6

***How To Use***
- Pull Project 
- Open with VS-Code or Android Studio 
- run ```flutter pub get```
- Run with browser 


***How To Build***
- Build web with ```flutter build web```
